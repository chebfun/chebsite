 DIAG(F)   Diagonal of a CHEBFUN2.
    G = DIAG(F) returns the CHEBFUN representing g(x) = f(x, x).
 
    G = diag(F,C) returns the CHEBFUN representing g(x) = f(x, x+c).
 
  See also TRACE.
