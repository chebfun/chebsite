<pre class="help-text"> GET   GET method for CHEBFUN2 class.
    P = GET(F, PROP) returns the property P specified in the string PROP from
    the CHEBFUN F. Valid entries for the string PROP are:
     'DOMAIN'
     'COLS'
     'ROWS' 
     'PIVOTVALUES'
     'PIVOTLOCATIONS'
