<pre class="help-text"> ISREAL   Real-valued CHEBFUN2 test.
    ISREAL(F) returns logical true if F does not have an imaginary part and
    false otherwise.
   
    ~ISREAL(F) detects CHEBFUN2 object that have an imaginary part even if it is
    all zero.
