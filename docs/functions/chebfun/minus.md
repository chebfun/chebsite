<pre class="help-text"> -   CHEBFUN minus.
    F - G subtracts CHEBFUNs F and G, or a scalar from a CHEBFUN if either F or
    G is a scalar.
 
    H = MINUS(F, G) is called for the syntax 'F - G'.
