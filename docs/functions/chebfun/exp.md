 EXP   Exponential of a CHEBFUN.
    EXP(F) computes the exponential of the CHEBFUN F.
 
    EXP(F, PREF) does the same but uses the CHEBFUNPREF object PREF when computing
    the composition.
 
    See also EXPM1.
