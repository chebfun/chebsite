 ACOS   Inverse cosine of a CHEBFUN.
    ACOS(F) computes the inverse cosine of the CHEBFUN F.
 
    ACOS(F, PREF) does the same but uses the CHEBFUNPREF object PREF when
    computing the composition.
 
  See also COS, ACOSD.
