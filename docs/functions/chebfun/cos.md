 COS   Cosine of a CHEBFUN.
    COS(F) computes the cosine of the CHEBFUN F.
 
    COS(F, PREF) does the same but uses the CHEBFUNPREF object PREF when
    computing the composition.
 
  See also ACOS, COSD.
