<pre class="help-text"> COT   Cotangent of a CHEBFUN.
    COT(F) computes the cotangent of the CHEBFUN F.
 
    COT(F, PREF) does the same but uses the CHEBFUNPREF object PREF when
    computing the composition.
 
  See also ACOT, COTD.
</pre>
