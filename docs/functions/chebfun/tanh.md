 TANH   Hyperbolic tangent of a CHEBFUN.
    TANH(F) computes the hyperbolic tangent of the CHEBFUN F.
 
    TANH(F, PREF) does the same but uses the CHEBFUNPREF object PREF when
    computing the composition.
 
  See also ATANH.
