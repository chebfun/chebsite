 ASEC   Inverse secant of a CHEBFUN.
    ASEC(F) computes the inverse secant of the CHEBFUN F.
 
    ASEC(F, PREF) does the same but uses the CHEBFUNPREF object PREF when
    computing the composition.
 
  See also SEC, ASECD.
