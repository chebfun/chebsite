<pre class="help-text"> SECH   Hyperbolic secant of a CHEBFUN.
    SECH(F) computes the hyperbolic secant of the CHEBFUN F.
 
    SECH(F, PREF) does the same but uses the CHEBFUNPREF object PREF when computing
    the composition.
 
  See also ASECH.
</pre>
