<pre class="help-text"> SINH   Hyperbolic sine of a CHEBFUN.
    SINH(F) computes the hyperbolic sine of the CHEBFUN F.
 
    SINH(F, PREF) does the same but uses the CHEBFUNPREF object PREF when computing
    the composition.
 
  See also ASINH.
</pre>
