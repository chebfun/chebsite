<pre class="help-text"> SIND   Sine of a CHEBFUN, result in degrees.
    SIND(F) computes the sine (in degrees) of the CHEBFUN F.
 
    SIND(F, PREF) does the same but uses the CHEBFUNPREF object PREF when computing
    the composition.
 
  See also ASIND, SIN.
</pre>
