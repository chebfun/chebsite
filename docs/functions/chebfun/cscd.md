 CSCD   Cosecant of a CHEBFUN, result in degrees.
    CSCD(F) computes the cosecant (in degrees) of the CHEBFUN F.
 
    CSCD(F, PREF) does the same but uses the CHEBFUNPREF object PREF when
    computing the composition.
 
  See also ACSCD, CSC.
