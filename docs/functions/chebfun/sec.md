 SEC   Secant of a CHEBFUN.
    SEC(F) computes the secant of the CHEBFUN F.
 
    SEC(F, PREF) does the same but uses the CHEBFUNPREF object PREF when computing
    the composition.
 
  See also ASEC, SECD.
