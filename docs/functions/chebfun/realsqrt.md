 REALSQRT   Real square root of a CHEBFUN.
    REALSQRT(X) is the square root of the CHEBFUN of X.  An error is produced 
    if X is negative or complex.
 
  See also CHEBFUN/SQRT, CHEBFUN/REALLOG, REALSQRT.
