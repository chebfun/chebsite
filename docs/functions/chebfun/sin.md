 SIN   Sine of a CHEBFUN.
    SIN(F) computes the sine of the CHEBFUN F.
 
    SIN(F, PREF) does the same but uses the CHEBFUNPREF object PREF when computing
    the composition.
 
  See also ASIN, SIND.
