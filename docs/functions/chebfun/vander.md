<pre class="help-text"> VANDER   Vandermonde array-valued CHEBFUN.
    A = VANDER(F, N) returns a Vandermonde array-valued CHEBFUN whose N columns
    are powers of the CHEBFUN F, that is A(:,j) = F.^(N-j), j = 0...N-1.
</pre>
