 ACSC   Inverse cosecant of a CHEBFUN.
    ACSC(F) computes the inverse cosecant of the CHEBFUN F.
 
    ACSC(F, PREF) does the same but uses the CHEBFUNPREF object PREF when
    computing the composition.
 
  See also CSC, ACSCD.
