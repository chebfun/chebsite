 SECD   Secant of a CHEBFUN, result in degrees.
    SECD(F) computes the secant (in degrees) of the CHEBFUN F.
 
    SECD(F, PREF) does the same but uses the CHEBFUNPREF object PREF when computing
    the composition.
 
  See also ASECD, SEC.
