 SEMILOGX   Semi-log scale plot of a CHEBFUN.
    SEMILOGX(...) is the same as PLOT(...), except a logarithmic (base 10) scale
    is used for the X-axis.
 
  See also PLOT, SEMILOGY, LOGLOG.
