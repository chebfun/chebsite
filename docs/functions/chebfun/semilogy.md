 SEMILOGY   Semi-log scale plot of a CHEBFUN.
    SEMILOGY(...) is the same as PLOT(...), except a logarithmic (base 10) scale
    is used for the Y-axis.
 
  See also PLOT, SEMILOGX, LOGLOG.
