<pre class="help-text"> >   Greater than operator for CHEBFUN objects.
    H = F > G, where F and/or G are CHEBFUN objects, constructs a logical
    CHEBFUN H which is true (i.e., takes the value 1) where F > G, and false (0)
    elsewhere.
 
  See also LT, GE, LE.
</pre>
