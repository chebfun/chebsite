 COTH   Hyperbolic cotangent of a CHEBFUN.
    COTH(F) computes the hyperbolic cotangent of the CHEBFUN F.
 
    COTH(F, PREF) does the same but uses the CHEBFUNPREF object PREF when
    computing the composition.
 
  See also ACOTH.
