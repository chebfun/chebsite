<pre class="help-text"> ISEQUAL   Equality test for two CHEBFUNs.
    ISEQUAL(F, G) returns logical 1 (TRUE) if the CHEBFUN objects F and G
    contain identical breakpoints and FUNS, and logical 0 (FALSE) otherwise.
 
  See also EQ.
</pre>
