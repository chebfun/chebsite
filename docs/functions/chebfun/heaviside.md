<pre class="help-text"> HEAVISIDE   Heaviside function of a CHEBFUN.
    HEAVISIDE(F) returns a CHEBFUN which is -1 when F < 0, +1 when F > 0, and
    0.5 when F == 0.
 
  See also DIRAC, SIGN.
</pre>
