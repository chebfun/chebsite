 LOGLOG  log-log scale plot of a CHEBFUN.
    LOGLOG(...) is the same as PLOT(...), except a logarithmic (base 10) scale
    is used for the X- and Y-axes.
 
  See also PLOT, SEMILOGX, SEMILOGY.
