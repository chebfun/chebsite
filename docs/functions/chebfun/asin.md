<pre class="help-text"> ASIN   Inverse sine of a CHEBFUN.
    ASIN(F) computes the inverse sine of the CHEBFUN F.
 
    ASIN(F, PREF) does the same but uses the CHEBFUNPREF object PREF when
    computing the composition.
 
  See also SIN, ASIND.
</pre>
