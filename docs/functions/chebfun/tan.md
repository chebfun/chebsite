 TAN   Tangent of a CHEBFUN.
    TAN(F) computes the tangent of the CHEBFUN F.
 
    TAN(F, PREF) does the same but uses the CHEBFUNPREF object PREF when
    computing the composition.
 
  See also ATAN, TAND.
