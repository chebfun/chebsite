 ATAN   Inverse tangent of a CHEBFUN.
    ATAN(F) computes the inverse tangent of the CHEBFUN F.
 
    ATAN(F, PREF) does the same but uses the CHEBFUNPREF object PREF when
    computing the composition.
 
  See also TAN, ATAND.
