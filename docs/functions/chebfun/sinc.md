 SINC   Sinc function of a CHEBFUN.
    SINC(F) computes the sinc function of the CHEBFUN F, i.e., sin(pi*F)/(pi*F).
 
    SINC(F, PREF) does the same but uses the CHEBFUNPREF object PREF when computing
    the composition.
 
  See also SIN.
