<pre class="help-text"> CSC   Cosecant of a CHEBFUN.
    CSC(F) computes the cosecant of the CHEBFUN F.
 
    CSC(F, PREF) does the same but uses the CHEBFUNPREF object PREF when
    computing the composition.
 
  See also ACSC, CSCD.
</pre>
