 RANK   Rank of an array-valued CHEBFUN.
    RANK(A) produces an estimate of the number of linearly independent columns
    (or rows) of A.
 
    RANK(A, TOL) is the number of singular values of A greater than TOL.
 
  See also SVD, QR.
