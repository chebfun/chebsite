 COSD   Cotangent of a CHEBFUN, result in degrees.
    COSD(F) computes the cotangent (in degrees) of the CHEBFUN F.
 
    COSD(F, PREF) does the same but uses the CHEBFUNPREF object PREF when
    computing the composition.
 
  See also ACOTD, COT.
