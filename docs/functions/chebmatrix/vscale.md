 VSCALE   Vertical scale of components in a CHEBMATRIX.
    VSCALE(A) returns a matrix containing the 'vertical scales' of the
    entries of a CHEBMATRIX as defined below:
        CHEBFUN: CHEBFUN/VSCALE()
        DOUBLE:  ABS()
        OPERATOR: NAN
