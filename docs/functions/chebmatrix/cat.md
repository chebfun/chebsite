 CAT   Concatenation of CHEBMATRIX objects.
    C = CAT(dim, A, B,...) concatenates chebmatrices along the indicated
    dimension following Matlab conventions.
 
  See also CHEBMATRIX.HORZCAT, CHEBMATRIX.VERTCAT.
